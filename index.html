Project: Server Creation with Proxmox

Class: 3°E Computer Science - I.T.I.S. Pininfarina

School Year: 2024–2025

Introduction

During the course, we learned how to install and configure a server using Proxmox VE, an open-source virtualization platform based on Debian.

Project Objectives

  • Create a virtual machine using Proxmox
  • Install an operating system inside the VM
  • Set up an FTP server and a web server
  • Publish an HTML page using FileZilla

Description

We created a server using Proxmox on a physical host machine. Inside it, a virtual machine with Ubuntu Server was launched. FTP and Apache services were installed on it, and an HTML website was uploaded via FileZilla.

Results

The server works correctly. The HTML page can be accessed via a browser, and contents can be updated through an FTP client.

Conclusion

This project allowed us to gain hands-on experience in server management and network service configuration.
